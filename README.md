# AgentImage 🖼️🖼️🚀

Agent Image is an AI-powered tool that transforms natural language prompts into structured image datasets.

## Features

* Chat-based interface
* Automatic image retrieval
* AI-powered query generation
* CSV dataset export
* Dynamic metadata enrichment (e.g. GPS, year built)

## How it works

1. User enters a prompt
2. AI generates search queries
3. Images are fetched and downloaded
4. User can request additional data fields
5. Dataset is enriched and exported

## Tech Stack

* FastAPI (backend)
* React (frontend)
* OpenAI / OpenRouter (LLM)
* Microsoft Bing API (images)
* Pexels API (images)
